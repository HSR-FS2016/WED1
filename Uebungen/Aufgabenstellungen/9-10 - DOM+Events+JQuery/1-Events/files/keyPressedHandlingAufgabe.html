KeyPress Event Handling Aufgabe

Die Buchstaben im Textfeld sollen stets als Uppercase dargestellt werden. Aktuell funktioniert dies nicht korrekt. Lösen Sie das Problem auf zwei Arten (es ist ok wenn nur das Anfügen von Zeichen funktioniert: Cursor in der Mitte des Strings muss nicht funktionieren)
  • Indem Sie einen anderen Event-Handler nutzen
  • Indem Sie die mittels Timeout einen neuen Event-Handler in der Event-Queue platzieren. Erst bei dessen Ausführung wird der String auf Uppercase geändert
